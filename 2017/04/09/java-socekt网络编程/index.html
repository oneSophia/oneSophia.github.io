java socekt网络编程

发表于 2017-04-09 |

　　今天把java的socket网络编程总结一下，也是为之后的网络编程深入学习留个参考。

一、客户端和服务器端的通信

实现的功能：客户端将信息发送给服务器端，服务器端接受到信息后，将相同的信息发送到客户端。
实现步骤：
服务器端：

  1. 创建SocketServer对象，绑定侦听端口
  2. 通过accept方法侦听客户端请求，获得一个socket
  3. 建立连接后，通过输入流接收客户端发送的信息
  4. 通过输出流发送相同信息到客户端
  5. 关闭资源
    客户端：
    1.创建Socket对象，指明需要连接的服务器端的ip地址和端口号
    2.建立连接后，通过输出流向服务器端发送信息
    3.通过输入流接收服务器端发送回来的信息
    4.关闭资源

二、多线程服务器端

实现功能：应用多线程实现客户端和服务器端的通信，服务器端一直侦听，当有客户端连接时，启动一个对应的线程和Socket；建立连接后，客户端将信息发送给服务器端，服务器端接受到信息后，将相同的信息发送到客户端。
实现步骤：
服务器端：

  1. 服务器端创建SocketServer，侦听端口
  2. 循环调用accept()等待客户端连接
  3. 连接成功一个客户端后，启动对应线程
  4. 获取输入流，打印客户端发来的信息
  5. 获取输出流，发送到客户端相同的信息
  6. 关闭相关资源
    这里的最主要功能放在了线程的run()方法中。
