java socket应用之文件上传器

发表于 2017-04-12 |

　　感觉Socket威力特别大，现在根据所学的知识，使用JDBC+Socket+IO+MySql等技术实现了一个控制台版的“文件上传器”，实现文件上传功能。　
　　主要功能：实现用户注册，用户登录，上传文件。
　　环境搭建：Eclipse+SQLyog

实现操作分析：

１.　用户注册

客户端选择“用户注册”，提示输入用户名、密码、确认密码，校验成功后将用户信息保存到数据库中，并提示用户注册成功，请登录。

２.　用户登录

客户端选择“用户登录”后提示输入用户名、密码，验证成功则提示上传文件，验证失败则给出提示信息，并可以重新登录。

３.　文件上传

客户端输入上传文件的路径，并读取文件数据，通过输出流发送到服务端，服务端接收后将上传文件保存到数据库中。

应用模型

　　本项目采用经典的MVC模式。模型层存储用户登录信息（用户名和密码），控制层：控制操作持久化层（模型层），控制操作数据库（DAO类），为视图层提供方法。视图层：和用户进行交互的页面
